# Sonic Main Portal

This repo is all about sonic main portal to get started with new wireframe.

```bash
Development Branch is 'dev'
```

## Run Locally

Clone the project

```bash
  git clone <repo_path>
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
